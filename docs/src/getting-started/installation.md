# Installation

## Using Cargo (Recommended)

If you have Rust installed, you can install Husk using Cargo:

```bash
cargo install husk-cli
```

## Verify Installation

After installation, verify that Husk is working:

```bash
husk --version
```

## Editor Setup

For the best development experience, install the VS Code extension or configure your editor with LSP support. See [Editor Setup](../reference/editor-setup.md) for details.
